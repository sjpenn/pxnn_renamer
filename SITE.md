# PxNN Renamer (Precision Slate)

## 1. Vision
PxNN Renamer is a high-performance utility for music professionals. The goal is to maximize metadata accuracy with minimal manual intervention. The interface should feel like a "precision instrument" — sterile, clean, and highly functional.
