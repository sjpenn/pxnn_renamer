# Design System: PxNN Renamer (Precision Slate)
**Project ID:** 7726411999300532473

## 1. Visual Theme & Atmosphere
Professional, data-focused, and clinically clean. Uses a layered slate and blue palette for a "Precision Instrument" feel.

## 3. Typography
- **Headlines:** `Manrope` (Weights: 600, 700, 800)
- **Body:** `Inter` (Weights: 400, 500, 600)
- **Icons:** Material Symbols Outlined

## 4. Layout
- Desktop-first, responsive grid.
- Uses `surface-container-high` for major panels.
- Pill buttons and rounded containers (`rounded-2xl` or `3xl`).
